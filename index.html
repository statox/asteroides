Asteroids... with real pew pews

This is my implementation of the well known game Asteroids. So you know the drill:

  • Right Arrow and Left Arrow to turn the spaceship
  • Up Arrow to turn the engine on
  • and Spacebar to shoot at the asteroids.

Once in a while you get a bonus or a malus, like autoshoot or slow turning.

And you have a button to turn the sound off and on, but honestly the sound effects are the main interest of this implementation


Close this window to start playing.


Warning for Firefox users: This greatly saddens me but the performances of this page on Firefox are terrible. This is because the library I'm using (p5.js) doesn't play well with a default setting privacy.resistFingerprinting.randomDataOnCanvasExtract of firefox.

If you know what you are doing you can change that in about:config, otherwise it's fine keep using Firefox this is a great browser and the net needs it
